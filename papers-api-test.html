üîç Advanced Query Parameters

status: Filter by publication status
Values: published, under-review, in-preparation, draft
Example: /api/papers?status=published
category: Filter by research category
Values: Infectious Disease, Global Health, Vector-borne Disease, etc.
Example: /api/papers?category=Global Health
author: Search by author name (partial match)
Searches both primary and co-authors
Example: /api/papers?author=Johnson
institution: Filter by institution (partial match)
Example: Harvard, WHO, CDC
Example: /api/papers?institution=Harvard
research_type: Filter by research methodology
Values: observational, meta-analysis, survey, surveillance, modeling, etc.
Example: /api/papers?research_type=meta-analysis
search: Full-text search across title, abstract, and keywords
Case-insensitive search
Example: /api/papers?search=malaria prevention
limit: Number of results to return (pagination)
Default: all papers
Example: /api/papers?limit=5
offset: Number of results to skip (pagination)
Default: 0
Example: /api/papers?limit=3&offset=3
sort_by: Field to sort by
Values: submission_date, title, author, citations, publication_date
Example: /api/papers?sort_by=citations
sort_order: Sort direction
Values: asc, desc (default: desc)
Example: /api/papers?sort_by=title&sort_order=asc
